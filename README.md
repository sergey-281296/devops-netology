## Игнорируемые файлы в .gitignore

В корневом .gitignore я буду добавлять общие правила для игнорирования.
В папке terraform/.gitignore будут игнорироваться:
- Директории .terraform/
- Файлы с расширениями .tfstate и .tfstate.*
- Файлы crash.log
- Файлы .tfvars
- Файлы override.tf, override.tf.json и файлы с суффиксом _override.tf
- Файлы .terraform.lock.hcl
